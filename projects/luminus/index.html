Luminus

May 2014
Game Programmer with The MecWarriors

About

The company Faceshift asked me to create a tech demo for their new facial recognition software that would be on display at a tech conference. I had just over one week to put together a small team, design and create a game based on recognizing player facial expressions, and submit the finished product before the conference began. It was a tight schedule, but we did it!

The resulting game Luminus was about capturing fireflies using only a set of simple facial expressions. Each different color of firefly the player encountered was connected to a different expression, and even the game menus could be used in the same way! This is one of the more unique projects I’ve worked on, and it was a great challenge to put together.
