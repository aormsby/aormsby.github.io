Github Action: Hugo Build and Deploy

Sep 2020

Download Star aormsby

About

This is an action for Hugo websites. Build your site from whichever repo branch you choose and deploy to a GH-Pages repo. This action supports deploying to the main site repo or to a repo submodule. No extra work is needed for submodule deploys. It’s intended use is to simply automate Hugo site builds.

I’m using this to deploy the site you are reading now.
