About

As a Game Design Intern at Backflip Studios, I assisted with multiple mobile game projects in the summer before my graduation year. I had a lot of fun redesigning levels for Ragdoll Blaster 3: Deluxe HD, and I was able to learn a lot through my work balancing some smaller game features in the hit game Paper Toss Friends. I also re-scaled Graffiti Ball levels for iPads and did some rapid prototyping for nascent concepts.
